# CC-DungeonExplorer

Explorer is a ([ComputerCraft][ccwiki]) Turtle program, atempting to explore and light up structures (as Mineshafts, dungeons) to prevent mob spawning.

## Mods compatible
* RogueLike
* Iron chest
* Chisel
